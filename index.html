Changing the logo and icon.

If you need to change the logo and icon that appears following the site title on each tab, as shown in the image below. Follow the steps below:





There are two ways to do this, as we all like simple things so I'm going to present here an easier way to do it.

1- Let's start with the logo which is the image that appears inside the site in the sidebar of menus, you will find it in the following path: spotify -> assets -> images -> logo.jpg The logo.jpg file is what you need to replace with another one with the same name and extension "logo.jpg" and with a minimum dimension: 200x200
just comply with this, you will already have the site logo changed.



2- Now it's time for the icon that appears in a tab following the site's title. To find it, you go to: spotify -> assets -> images -> icons -> icon.ico Pay attention here because the extension of your icon should be '.ico' and don't worry if you don't know where to get it, the site https://convertio.co/ will help you convert an image file '.png' by example for the desired '.ico'. As in the previous step, you need to replace the old "icon.ico" file with yours with the same name and extent.

Creating a database in cPanel

1- Click MySQL Databases under Databases in cPanel.





2- Under Create New Database, enter the name of the database in the New Database:





3- When ready, click Create Database.

The new database is created and ready to use. You will also find the new database added to the Current Databases table.





In order to be able to fill and maintain the database, you need to create and assign a MySQL user to it.



4- Under MySQL Users > Add New User, enter the MySQL username in the Username text box.





Once you have added the new user, it will be added to the Current Users table.



5- In the Password text box, enter the user password.

6- In the Password (Again) text box, retype the user password.

7- When ready, click Create User

8- Under Add User to Database, select the user that you want to add from the User drop-down menu.





9- In the Database drop-down menu, select the database.





10- Click Add.

11- Select the privileges you want to grant the user, or click ALL PRIVILEGES to grant the user all permissions to the database.





12- When ready, click Make Changes.





Thatâ€™s it! You can now see the user assigned to the database in the Current Databases table.





Inserting datas into the created database

1- Click phpMyAdmin under Databases in cPanel.





2- Any databases you have will appear here. Look for the one you created in the previous section





3- Select the previous created database.





4- Click on SQL tab.





5- Open the main file to find the data that we have prepared to make your work easier. Follow the following path: spotify -> assets -> database_import -> tubiancanga.sql





6- Open this file with a text editor that you have on your computer and once it is open, copy all the content into this file and let's go back to phpMyAdmin.



7- Now let's paste the content that we copied in the previous step, in the text space that the SQL tab gave us when we opened it in step 4.





7- Click Run.





8- This will be the result showing that you were successful, congratulations on getting here, you are a winner.

Uploading our web aplication.

1- Click File Manager under Files in cPanel.





2- Double click on "public_html" to open that file as this is where your site should be installed.





3- Click Upload.

If you find any files in this folder, feel free to delete them or just leave them alone, it won't be a problem.





4- Click Select File.





5- The zipped file that you must upload is already prepared, open the main folder and follow the following path: spotify -> assets -> zip_project -> spotify.zip





6- Select the zipped file and click open





6- Wait for the upload process to finish.





7- Once the process is complete, click back to...





8- Click on the zipped file to select it and then click on extract





9- Click extract file to confirm unzipping.

Quick start

If you want to use this theme in a web aplication, you need to Copy and paste the folders: css , js, fonts into the application's root folder, and follow the steps bellow:

Copy-paste the stylesheet "link" into the "head" of the page where you want to include the forms, after all other stylesheets to load the theme's CSS.

	
	<!-- Theme's CSS files ============================================ -->
		<link rel="stylesheet" href="css/style.css">
		<link rel="stylesheet" href="css/font-awesome.min.css">

At the end of your signin and signup page, right after the closing "body" tag. Be sure to place js first. 

	
	<!-- Theme's JS file ============================================ -->
        <script src="js/script.js"></script>

Copy-paste the struture of the div container of the theme's html files into the pages of the application See the struture of div container in the section HTML Struture.
